# 賭けボーリングアプリ 🎳

チーム戦ボーリングの賭けポイント計算アプリ

## 📦 含まれるファイル

- `bowling-bet.html` - メインのアプリファイル
- `manifest.json` - PWA設定ファイル
- `sw.js` - Service Worker（オフライン対応）
- `icon-192.png` - アプリアイコン（小）
- `icon-512.png` - アプリアイコン（大）

## 🚀 デプロイ方法

### 方法1: Netlify（一番簡単！）

1. https://app.netlify.com/drop にアクセス
2. すべてのファイルをドラッグ&ドロップ
3. 自動的にURLが発行されます（例: https://your-app.netlify.app）
4. URLを共有すればみんなが使えます！

### 方法2: Vercel

1. https://vercel.com にアクセス
2. GitHubアカウントでログイン
3. 「New Project」→ ファイルをアップロード
4. デプロイ完了！

### 方法3: GitHub Pages

1. GitHubにリポジトリを作成
2. すべてのファイルをアップロード
3. Settings → Pages → Source を main/master ブランチに設定
4. https://your-username.github.io/your-repo でアクセス可能

### 方法4: Cloudflare Pages

1. https://pages.cloudflare.com にアクセス
2. アカウント作成（無料）
3. ファイルをアップロード
4. 自動的にURLが発行されます

## 📱 使い方

### ホーム画面に追加

**iPhone（Safari）**
1. SafariでアプリのURLを開く
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

**Android（Chrome）**
1. ChromeでアプリのURLを開く
2. 右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

これでアプリのようにホーム画面から起動できます！

## ✨ 機能

- ✅ 1ピンあたりのポイント設定（自由に変更可能）
- ✅ プレイヤー名の変更・追加・削除
- ✅ チーム戦対応（チームA vs チームB）
- ✅ 自動計算機能
- ✅ 勝敗判定とポイント差表示
- ✅ スマホ対応
- ✅ オフラインでも動作（PWA対応）
- ✅ ホーム画面に追加可能
